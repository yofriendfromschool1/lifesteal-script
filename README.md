# lifesteal-script
script to setup lifesteal minecraft on 1.21.10
linux btw
```
git clone https://github.com/yofriendfromschool1/lifesteal-script.git
cd lifesteal-script
chmod +x *.sh
./runfabric\ installer.sh
./movetoit.sh
```
for windows users:
just download everything and double click fabric installer and install 1.21.10 then move all other jar files in %appdata%\.minecraft\mods
